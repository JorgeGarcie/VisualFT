# ExecPlan Template

Copy this template to `docs/exec-plans/active/<plan-name>.md` when starting a new plan.

---

```markdown
# ExecPlan: <Title>

**Status**: Draft | In Progress | Blocked | Complete
**Created**: YYYY-MM-DD
**Author**: <name>
**Motivated by**: <link to experiment or issue if applicable>

## Goal

One sentence: what does this change accomplish?

## Context

Why is this needed now? What problem does it solve?

## Plan

### Step 1: <description>
- [ ] Task A
- [ ] Task B

### Step 2: <description>
- [ ] Task C

## Progress

| Date | Update |
|------|--------|
| YYYY-MM-DD | Started step 1 |

## Decision Log

| Decision | Rationale | Date |
|----------|-----------|------|
| Chose X over Y | Because Z | YYYY-MM-DD |

## Surprises

Things that didn't go as expected:
- (none yet)

## Outcomes

**Result**: (filled when complete)
**Follow-ups**: (any new plans or debt items spawned)
```

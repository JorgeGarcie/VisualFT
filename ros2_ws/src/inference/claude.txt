  ---                                                                                                                              
  Task: Build a ROS2 node for real-time tendon classification inference.                                                         
                                                                                                                                   
  Files location on Flexiv computer: ~/tendon_classifier_inference/                                                                
  - best.pth — model weights
  - spatial_combined.yaml — model config
  - config.py, models_v2.py, encoders.py, attention.py — model code

  Subscribes to:
  - /camera/image_raw (sensor_msgs/Image)
  - /coinft/wrench (geometry_msgs/WrenchStamped)

  Publishes:
  - /tendon_class (std_msgs/Int32) — 0=none, 1=single, 2=crossed, 3=double
  - /tendon_class_name (std_msgs/String)

  Preprocessing:
  # Image: center crop to 1080x1080, resize to 224x224, ImageNet normalize
  CROP_SIZE = 1080
  MEAN = [0.485, 0.456, 0.406]

  # Force: [fx, fy, fz, tx, ty, tz] as float32 tensor (1, 6)
  # from wrench_msg.wrench.force.{x,y,z} and wrench_msg.wrench.torque.{x,y,z}

  Model loading:
